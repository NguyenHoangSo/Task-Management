# Task Management
## User authentication
### Register
1. User can register by enter a email and password
2. User can use a google account
### Login
1. Enter email and password
2. using previous google account
## Dashboard
1. After login, users are directed to the dashboard where they can view:
- Total number of tasks
- Number of completed tasks
- Number of overdue tasks
- Recently updated or created tasks
2. The bellow, a list of current tasks is displayed
## Task management
1. User can create and manage their tasks
- Each task conclude:
-- Title
-- Description/Content
-- Richtext
    - Bold 
    - Italic
    - Checkbox
    - Hyper link
    - Image
    - List
-- Status
-- Priority
-- Assigned person
-- Creator
-- Due date
2. User can perform these actions
- Create task
- Edit task
- Delete task
- Assign task to another user
- Share task with collaborators
## Task Status Workflow
1. During task execution, users can update task status:
- To Do
- Doing / In Progress
- Done
- This helps track progress clearly.
## Priority Management
1. Users can classify tasks by priority:
- High
- Medium
- Low
2. This helps users focus on urgent or important work first.
## Search / Filter / Sort
1. Users can organize tasks using:
-- Filter by:
- Due date
- Priority
- Status
- Assigned user
-- Sort by:
- Deadline
- Priority level
- Creation date
## Collaboration
1. The application supports teamwork through:
- Sharing tasks with others
- Commenting on tasks
- Viewing updates from collaborators
## Activity History
1. Users can review:
- Daily task history
- Completed actions
- Recent edits
2. Useful for tracking productivity progress.
## Calendar View
1. Users can monitor tasks through a calendar / timetable interface.
2. This helps them visualize:
- Deadlines
- Daily workload
- Upcoming tasks
